# Aliya Sayed - Personal Portfolio

A modern and interactive personal portfolio website showcasing my professional journey, skills, and experience. Built with React, TypeScript, and SCSS modules.

## Features

- Responsive and modern design
- Interactive UI components
- Smooth scroll animations
- Dynamic skill progress bars
- Timeline-based experience and education sections
- Contact form with validation
- Social media integration
- Loading animations and transitions
- Mobile-first approach

## Project Structure

### Components
- `Navbar`: Responsive navigation menu
- `Background`: Dynamic background effects
- `Photo`: Profile photo component
- `Loader`: Loading animations
- `Tags`: Skill and technology tags
- `SocialButtons`: Social media links
- `Location`: Location display component
- `Section`: Reusable section wrapper
- `DateDisplay`: Date formatting component
- `SkillProgressBar`: Animated skill progress indicators
- `Timeline`: Experience and education timeline
- `Button`: Reusable button component

### Sections
- `Hero`: Introduction and main banner
- `Skills`: Technical skills showcase
- `About`: Personal information and background
- `Education`: Academic background
- `Contact`: Contact form and information
- `Experience`: Professional experience timeline

### Additional Directories
- `services/`: API and service integrations
- `utils/`: Helper functions and utilities
- `context/`: React context providers
- `styles/`: Global styles and SCSS modules
- `templates/`: Reusable page templates

## Technologies Used

- React 19
- TypeScript
- SCSS Modules
- Vite
- React Router
- Framer Motion (for animations)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
```

2. Install dependencies:
```bash
npm install
# or
yarn
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
# or
yarn build
```

## Development

The project follows a component-based architecture with a focus on reusability and maintainability. Each component is styled using SCSS modules for scoped styling and better organization.

### Key Features Implementation

- **Responsive Design**: Mobile-first approach with breakpoints for different screen sizes
- **Animations**: Smooth transitions and loading effects using Framer Motion
- **Performance**: Optimized images and lazy loading for better performance
- **Accessibility**: Semantic HTML and ARIA attributes for better accessibility
